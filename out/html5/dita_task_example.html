DITA_Task_Example

This is an example of a DITA task topic

This outlines the steps in an example task

You will need to follow these steps to create a task

  1. In Oxygen XM, select File > New
  2. Select the task topic from the Framwork Template folder.
  3. Name your task topic
  4. Follow the file path to the folder where you will save this task topic
  5. Select Save
